# Robinhood Rug Analyzer

Production-ready project architecture for a token rug-risk analyzer. This version accepts a token contract address and detects the likely blockchain from address format only. Rug analysis is intentionally not implemented yet.

## Project Structure

```text
robinhood-rug-analyzer/
+-- app/
¦   +-- api/
¦   +-- core/
¦   +-- models/
¦   +-- services/
¦   +-- main.py
+-- frontend/
+-- logs/
+-- .env
+-- .gitignore
+-- README.md
+-- requirements.txt
```

## Folder Guide

- `app/` contains the FastAPI backend package.
- `app/api/` contains API routes and request handlers.
- `app/core/` contains configuration and logging setup.
- `app/models/` contains Pydantic schemas for request and response contracts.
- `app/services/` contains modular service logic, including blockchain detection stubs.
- `frontend/` contains the simple HTML, CSS, and JavaScript interface.
- `logs/` stores runtime log files generated by the backend.

## Setup

```bash
python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
uvicorn app.main:app --reload
```

Then open `http://127.0.0.1:8000`.

## Environment Variables

Copy or edit `.env` with API keys as needed:

```env
ETHERSCAN_API_KEY=""
BSCSCAN_API_KEY=""
POLYGONSCAN_API_KEY=""
```

## Current Scope

Implemented:
- FastAPI backend architecture
- Static frontend
- Contract address submission
- Basic blockchain format detection
- Environment-based settings
- Logging configuration

Not implemented:
- Rug-risk analysis
- External blockchain API calls
- Token liquidity, holder, honeypot, or ownership checks

## Free Test Deployment on Render

This project includes `render.yaml` for Render's free web service tier.

1. Push this folder to GitHub.
2. In Render, choose **New +** > **Blueprint**.
3. Connect the GitHub repository.
4. Render will read `render.yaml` and run:
   - Build: `pip install -r requirements.txt`
   - Start: `uvicorn app.main:app --host 0.0.0.0 --port $PORT`
5. Open the generated Render URL and test the frontend.

The API test endpoint is:

```text
POST /api/v1/analyze
```

Example body:

```json
{
  "contract_address": "0x0000000000000000000000000000000000000000"
}
```
